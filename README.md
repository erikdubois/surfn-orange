A colour variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the **Surfn-Orange** theme and inherits the base **Surfn** icon set, so `surfn-icons-git` is required.

## Installation (Arch / Kiro — nemesis_repo)

```bash
# Everything — the whole Surfn collection (base + all variants)
sudo pacman -S surfn-icons-meta

# Only this variant
sudo pacman -S surfn-orange-icons-git
```

`surfn-icons-git` (base Surfn) is pulled in automatically as a dependency.

## Manual

Copy the **Surfn-Orange** icon folder into `~/.icons` (or `~/.local/share/icons` on Plasma),
make sure the base `Surfn` theme is present, then select **Surfn Orange** in your appearance settings.
